# Multi-Object Tracking & Re-Identification System

Production-ready multi-object tracking system with person re-identification capabilities.

## Features

- Real-time object detection using YOLOv8
- Multi-object tracking with ByteTrack
- Person re-identification using ResNet50
- REST API with FastAPI
- Docker deployment support
- MLOps integration with MLflow and DVC

## Installation
```bash
poetry install
poetry shell
```
